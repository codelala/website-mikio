About Mikio Aoi

The bulk of my research has been focused on identifying new methods of data analysis that leverage knowledge of the data-generating model in order to have more sensitive and informative tools with which to do perform inference. In my graduate work, I did this for the analysis of cerebral autoregulation. I used a biophysical model to summarize the interactions between physical and regulatory parameters, and used this model in concert with clinical data to asses cerebral autoregulatory function in people suffering from chronic ischemic stroke. In my current work, I am developing metrics for the analysis of rhythmic phenomena in neural systems. By modeling the statistical processes that are assumed when using conventional metrics of rhythms and rhythmic synchrony, we can identify how small deviations from these assumptions influence those metrics and suggest analysis methods that might behave more favorably to those deviations.

I am currently developing spectral anlaysis tools for spike-field synchrony and instantaneous frequency estimation tools using time-frequency representations. I would like to learn more about computational neuroscience and about how (or why) the nervous system uses rhythms to perform computations.
